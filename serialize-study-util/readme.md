Честным людям МОЕ почтение!

Это проект, который представляет собой набор небольших программ, которые выполняют одну техническую задачу
Состав проекта

Пакет entity
Есть три вида сущностей которые используются для
 а) сериализации (serialize)
 б) ручной сериализации, через экстернализацию (externalize). Обратите на доп.методы в классе. По сути мы программируем поведение
 в) сгенерированные через XSD классы (jaxb)

Пакет da 
Функции, которые читают из/пишут в файл(а) переданную сущность. Так же разделены на три группы с учетом типов сущностей

Пакет util 
Выполняет основную работу по управлению потоком выполнения программы
Решается две задачи
 а) Считать данные из источника и с помощью механизмов сериализации сохранить промежуточное состояние
 б) Восстановить объект и преобразовать его в требуемы формат (xml, json)
